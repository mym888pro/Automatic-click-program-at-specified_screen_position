Automatic Click Program at Specified Screen Position 自动点击程序

A Python-based tool for automating mouse clicks at specified screen positions with a customizable frequency.
一个基于 Python 的工具，可以在指定屏幕位置自动点击，并可自定义点击频率。

Features 功能

  • Click automation at any given screen coordinates (X, Y).
    在任何指定的屏幕坐标（X, Y）位置自动点击。
  • Customizable frequency of mouse clicks.
    可自定义鼠标点击频率。
  • Click only when the mouse stops moving.
    只有当鼠标停止移动时才会点击。
  • Written in Python for easy modification and usage.
    使用 Python 编写，便于修改和使用。

Installation 安装

To get started, clone this repository and install the necessary dependencies.
开始使用前，克隆这个库并安装必要的依赖。

git clone https://github.com/your-username/Automatic-click-program-at-specified_screen_position.git
cd Automatic-click-program-at-specified_screen_position
pip install -r requirements.txt

Usage 使用方法

To use the program, simply run the Python script with the specified screen coordinates.
要使用此程序，只需运行带有指定屏幕坐标的 Python 脚本。

python auto_clicker.py --x 100 --y 200 --frequency 5
