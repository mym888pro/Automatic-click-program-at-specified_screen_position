屏幕指定位置自动点击程序

一个使用 Python 的 PyAutoGUI 库实现的屏幕指定位置自动点击程序。
可以在你指定的屏幕坐标上快速连续点击，支持 Windows 和 Android（qpython）。

Windows 使用说明

  • 安装必要库：
    pip install pyautogui keyboard
  • 运行程序后，会显示当前屏幕分辨率和鼠标位置。
  • 输入参数：
    X坐标（留空使用当前鼠标X位置）
    Y坐标（留空使用当前鼠标Y位置）
    点击次数（默认1000次）
    点击间隔（默认0.1秒）
  • 按 F6 键开始自动点击，按 ESC 键停止。
  • 确保目标窗口位于前台且不被遮挡。
  • 频繁点击可能被某些程序检测为异常。
  • 请遵守相关软件使用条款。

Android 使用说明（qpython）

  • 基于 qpython，使用 androidhelper 模块（QPython自带）。
  • txt 文件中的代码为 qpython 用 Python 脚本，请复制到 qpython 中运行。
  • 下载 qpython：Google Play 或应用商店。
注意事项：
  • 目标窗口需在前台且不被遮挡。
  • 频繁点击可能被某些程序检测为异常。
  • 使用自动化工具请遵守相关软件条款。
  • 程序运行期间可移动鼠标/点击屏幕，但不要干扰目标区域。
  • Android 上点击速度有限，最小间隔约 0.05 秒（20次/秒）。
  • 运行时需保持屏幕亮着且不锁屏。

扩展说明

  • 如需多个点击点、随机间隔等复杂功能，可自行扩展本程序。
  • 本程序无版权限制，自由使用。
  • 如需更专业的 Android 自动点击工具，可考虑 Auto.js（仅作推荐）。
